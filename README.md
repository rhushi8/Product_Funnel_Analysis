# E-Commerce Product Funnel Analysis

## Overview
This project analyzes user behavior across an e-commerce funnel to understand how users move from product views to purchases.

The analysis identifies conversion rates, category performance, and user activity patterns using SQL, Python, and Power BI.

---

## Tech Stack
SQL  
Python (Pandas, Matplotlib)  
Power BI  
Git & GitHub  

---

## Dataset
E-commerce user event dataset containing product views, cart additions, and purchases.

Due to GitHub size limitations, raw datasets are not included in this repository.

---

## Analysis

### Funnel Stages
View → Add to Cart → Purchase

Key metrics:

• Product Views: **587K**  
• Add to Cart: **20K**  
• Purchases: **17K**  
• Overall Conversion Rate: **~3%**

---

## Key Insights

• Only ~3% of product views convert into purchases  
• Cart → Purchase conversion is strong (~85%)  
• Electronics categories generate the highest revenue  
• User activity peaks during evening hours
